data usage

22/09 2025

at this point the goal was simply to minimise data usage of the website

previousely the website would load all images at full quality (all 1000 pictures of 11 mb) this will kill any mobile data connection

to fix this i made a sccript that generates an array going from 1-n where n is the number of pictures, this then gets randomly shuffled

now when displaying the images we load a small amount of images (10) at a time, and we add in a button to load the next pictures

on to of this the ability to unshuffle dthe pictures was added, and the ability to jump to a specific point in the array of pictures

lastly the pictures where replaced with thumnails that where significantly smaller
